## 🚀 Instalación y Configuración

### 📋 Prerrequisitos

Asegúrate de tener instalado:
- **Node.js** (versión 18.0 o superior)
- **npm** o **yarn**
- **Git**

### 🔧 Instalación

1. **Clona el repositorio**
   \`\`\`bash
   git clone https://github.com/tu-usuario/portfolio.git
   cd portfolio
   \`\`\`

2. **Instala las dependencias**
   \`\`\`bash
   npm install
   # o
   yarn install
   \`\`\`

3. **Ejecuta el proyecto en desarrollo**
   \`\`\`bash
   npm run dev
   # o
   yarn dev
   \`\`\`

4. **Abre tu navegador**
   
   Visita [http://localhost:3000](http://localhost:3000) para ver el resultado.

### 🏗️ Build para Producción

\`\`\`bash
npm run build
npm start
\`\`\`

## 🎨 Personalización

### 🎯 Configuración Básica

1. **Información Personal**
   
   Edita el archivo \`app/page.tsx\` y actualiza:
   \`\`\`typescript
   const personalInfo = {
     name: "Tu Nombre",
     title: "Tu Título Profesional",
     description: "Tu descripción personal",
     email: "tu@email.com"
   }
   \`\`\`

2. **Habilidades y Tecnologías**
   
   Modifica el array \`skills\` en \`app/page.tsx\`:
   \`\`\`typescript
   const skills = [
     { name: "React", icon: Code, color: "text-cyan-500" },
     // Añade más habilidades...
   ]
   \`\`\`

3. **Proyectos**
   
   Actualiza el array \`projects\` con tus proyectos:
   \`\`\`typescript
   const projects = [
     {
       title: "Mi Proyecto",
       description: "Descripción del proyecto",
       technologies: ["React", "Node.js"],
       image: "/ruta/a/imagen.jpg"
     }
   ]
   \`\`\`

### 🎨 Personalización de Estilos

- **Colores**: Modifica \`tailwind.config.ts\`
- **Fuentes**: Actualiza \`app/layout.tsx\`
- **Animaciones**: Edita \`app/globals.css\`
